# Laravel insan-Erapor web

Pada SD Insan Mulia pengarsipan nilai siswa masih menggunakan excel sebagai media penyimpanan nilai. Masalah yang terjadi dalam penggunaan sistem rapor berbasis Excel di SDIT Insan Mulia dapat menimbulkan sejumlah kendala, seperti kesulitan dalam pengelolaan data, kurangnya keamanan informasi, dan keterbatasan aksesibilitas bagi orang tua siswa. Dengan mengadopsi sistem e-rapor berbasis website, sekolah berharap dapat meningkatkan efisiensi dalam pengelolaan nilai, menyediakan akses yang lebih mudah bagi orang tua, dan menjaga keamanan serta integritas data siswa.
